## Tag Selection Guidelines

When recommending tags for an article:

1. **Primary topic tag**: The tag that best matches the article's main subject.
2. **Process tags**: Tags for tools/processes the article describes (e.g., `RightSignature Completed` if RightSignature is used).
3. **Ancillary tags**: Tags for secondary topics covered (e.g., `Taxes` if tax implications are discussed, `MFA` if MFA is relevant, `Participant Dashboard` if portal access is discussed).
4. **Limit**: Aim for 3-6 tags per article. Fewer than 3 may miss retrieval opportunities; more than 6 risks noise.
5. **Justify each tag**: The article must contain substantive content about the tag's topic, not just a passing mention.
